# Проект 1: MiniReduxApp — SPA с собственной архитектурой и стейт-менеджером

## 📌 Описание
Разработать одностраничное приложение (SPA) на нативном JavaScript, реализовав:
*   Архитектурный шаблон (MVC или MVVM)
*   Собственный аналог Redux (управление состоянием)
*   DOM-рендеринг компонентов через JS
*   Стилизацию полностью через JS (никаких `*.html`, `*.css`)
*   Навигацию по страницам без перезагрузки (роутер)

## 📋 Функциональность
*   Таск-трекер (или простой блог, или UI-интерфейс по выбору студента)
*   Добавление/удаление/редактирование сущностей
*   Интерактивные UI-элементы (кнопки, инпуты, чекбоксы)
*   Отображение состояния приложения
*   Рендер компонентов по подписке на стейт
*   Встроенный роутер (hash-based или history API)
*   Сохранение данных в `localStorage`

## ⚙ Что нужно реализовать самостоятельно

### ✅ Собственный аналог Redux
*   `createStore`, `getState`, `dispatch`, `subscribe`
*   Reducer-функции, `combineReducers`

### ✅ Архитектуру
*   Модули: `view`, `model`, `controller`
*   Реализация Router вручную (без библиотек)

### ✅ Отрисовку
*   Использование `document.createElement`, `appendChild`, `textContent`, `setAttribute`, `style`
*   Никаких HTML-файлов — всё через JS (включая `head`, `meta`, `body`, `layout`)

### ✅ Мини-фреймворк
*   Компонентная модель (например, `BaseComponent`, `mount`, `unmount`)
*   Обработка событий (`addEventListener`) вручную
*   Ререндер при обновлении состояния

### ✅ Стилизация
*   Создание style-блоков через JS или `element.style`
*   Минимальный UI: карточки, кнопки, формы — всё должно выглядеть приятно, опрятно и понятно, даже без CSS-файлов
*   Минимальный responsive: не full adaptive, но логика автоширины/высоты, flex-стили можно создать через `style.display = 'flex'`

### ✅ Тестирование
Покрытие тестами:
*   reducer'ов
*   бизнес-логики
*   утилит (`debounce`, `clone`, `deepEqual`)

## 📦 Технологии и сборка
*   Сборка через `npm` + `vite`/`webpack`
*   Код должен быть структурирован по папкам:
    ```text
    core, components, store, router, utils, tests
    ```
